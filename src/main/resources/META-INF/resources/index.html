Getting Started

            

                
Prerequisites

                
• Java 21 or higher
• Gradle 7.5+ (included via wrapper)
• Docker (optional, for containerization)


                
Clone and Run

                
# Clone the repository
git clone <repository-url>
cd quarkus-rest

# Run in development mode with live reload
./gradlew quarkusDev

# The application will be available at:
# http://localhost:8080


                
Build and Package

                
# Create a runnable JAR
./gradlew quarkusBuild

# Run the JAR
java -jar build/quarkus-app/quarkus-run.jar

# Build native executable (requires GraalVM)
./gradlew build -Dquarkus.package.type=native


                
Docker Support

                
# Build Docker image
docker build -f src/main/docker/Dockerfile.jvm -t quarkus-rest .

# Run with Docker
docker run -i --rm -p 8080:8080 quarkus-rest

• Hibernate ORM 6.6.17 - Object-relational mapping

• OpenTelemetry 1.44.1 - Distributed tracing

• RESTEasy Reactive - JAX-RS implementation
• Jackson 2.18.3 - JSON processing
• Swagger UI - API documentation
• WebSockets - Real-time communication

• Docker support built-in
• GraalVM native compilation
• Kubernetes integration

• Caffeine 3.2.0 - High performance caching
• gRPC 1.69.1 - High performance RPC framework
